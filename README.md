# epubify

把小说类 TXT 文本转换为 EPUB 3 的小工具，自动识别书名、作者、简介与卷/章目录。

## 功能概览
- 自动检测编码（UTF-8 / GB18030）
- 识别书名、作者、简介（标签式与非标签式）
- 识别卷/章/常见特殊章节并生成目录
- 生成 EPUB 3（含 `nav.xhtml` 与 `toc.ncx`）

## 运行环境
- Python 3.12+

## 用法
```bash
python3 epubify.py "输入.txt"
```

指定输出路径：
```bash
python3 epubify.py "输入.txt" -o "输出.epub"
```

## 目录规则（简述）
- `第x卷` 作为一级目录项
- `第x章` / `第x节` / `第x回` / `Chapter N` 作为章节
- `序章` / `楔子` / `后记` / `番外` 等视为章节

## 说明
- 简介只写入 metadata 并显示在前置信息页，不单独作为正文章节
- 正文按“每行一段”处理
